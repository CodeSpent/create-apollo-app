## Spin.js - a JavaScript build tool that makes your life easier

[![Twitter Follow](https://img.shields.io/twitter/follow/sysgears.svg?style=social)](https://twitter.com/sysgears)

## Installation

```bash
npm install -g spinjs
```

## Usage

```bash
spin --help
```
